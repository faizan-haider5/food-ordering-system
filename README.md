## ✨ About the Project

This **Food Ordering System** is a C++ console application developed as part of the **CSI-302** course project. It simulates a real-world food ordering experience where users can:

- 🍕 Browse a restaurant menu
- 🛒 Add items to their cart
- 💳 Place and confirm orders
- 🧾 View order summary with total bill

The system is designed using **Object-Oriented Programming (OOP)** principles and file handling in C++, making it modular, readable, and easy to extend.

---

## 🍽️ Features

| Feature | Description |
|---|---|
| 📜 **Menu Display** | View categorized food items with prices |
| 🛒 **Cart Management** | Add, remove, and update items in your cart |
| 💰 **Billing System** | Auto-calculates total cost with a clean invoice |
| 🔐 **User Interaction** | Simple and intuitive menu-driven interface |
| 📂 **File Handling** | Persistent data storage using file I/O |

## ⚙️ Getting Started

### Prerequisites

Make sure you have a **C++ compiler** installed on your system:

- ✅ **GCC / G++** (Linux / Mac)
- ✅ **MinGW** (Windows)
- ✅ **Visual Studio** (Windows)
- ✅ **Code::Blocks / Dev-C++**

### Installation

```bash
# 1️⃣ Clone the repository
git clone https://github.com/faizan-haider5/food-ordering-system.git

# 2️⃣ Navigate into the project directory
cd food-ordering-system/Food\ Ordering\ System\ C++/Food\ Ordering\ C++
```

---

## 🚀 How to Run

### 🖥️ On Linux / Mac (Terminal)

```bash
# Compile the source
g++ Source.cpp -o FoodOrderingSystem

# Run the application
./FoodOrderingSystem
```

### 🪟 On Windows (Command Prompt)

```cmd
g++ Source.cpp -o FoodOrderingSystem.exe
FoodOrderingSystem.exe
```

### 🔵 Using Visual Studio / Code::Blocks

1. Open the project in your IDE
2. Add `Source.cpp` to the project
3. Hit **Build & Run** ▶️
